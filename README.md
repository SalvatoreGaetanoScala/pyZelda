# 🗡️ pyZelda

**pyZelda** è un gioco Zelda-like sviluppato interamente in Python. 

Attualmente il progetto si trova in **fase Alpha**: il gioco è ancora nelle prime fasi di sviluppo (un po' acerbo) e potresti incontrare qualche bug tecnico. 
---

## 📋 Requisiti di Sistema

Per poter giocare o modificare il codice, assicurati di avere installato:
* **Python**: versione **3.12** o superiore. ([Scarica Python qui](https://www.python.org/downloads/release/python-3120/))
* **Pygame**: la libreria principale su cui si basa il motore del gioco.
